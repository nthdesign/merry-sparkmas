This year, the most exciting item on my Christmas wish list was a reward from a Kickstarter project I backed months ago. Santa brought me a Spark Core from @sparkdevices. This cool little gadget is, essentially, an Arduino with built-in WiFi. The best part is, the developers at Spark built an API that lets me send REST requests to this device from anywhere in the world!

My first project with the Spark Core, titled "Merry Sparkmas," is an ornament that displays holiday greetings sent from this very webpage. Type a message in the box below, optionally add your Twitter username, and press the Send Holiday Greetings button. Within seconds, the ornament will start to glow, sharing your holiday cheer. It's a Sparkmas miracle!
